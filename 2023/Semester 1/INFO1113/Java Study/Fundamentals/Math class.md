
Math class Functions: 

```java
Math.max(x,y) // return the max value 

Math.min(x,y) // return the min value 

Math.abs(x) // return absolute value 

Math.sqrt(x) // return square root value 

Math.pow(x, 3) // returns value to the power of 3

Math.round(x)  // returns the rounded value 

Math.ceil(x) // return rounded up value 

Math.floor(x) // returns rounded down value

```
